###  关于  
这是一个Java终端插件工程


###  环境配置   
*  支持eclipse3.7以上版本  

*  需要配置eclipse才能正常编译，将jars中的jar包放入eclipse安装目录下的“plugins”中，  
重启eclipse即可。  
  
  

###  编译   
将插件工程源码导入eclipse，clean  


###   运行   
Run as Eclipse Application   


###  预期  
window-show view中能找到Terminal；  
工具条中会多一个类似终端图标；
